# Quickstart for Basic Python Calculator

**Date**: 2025-12-02

## Running the Calculator

1.  Navigate to the `src` directory:
    ```bash
    cd src
    ```
2.  Run the `main.py` file:
    ```bash
    python main.py
    ```

## Running the Tests

1.  Navigate to the root directory of the project.
2.  Run pytest:
    ```bash
    pytest
    ```
